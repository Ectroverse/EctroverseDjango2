INSTALLATION
written by: ImNotHere


0) Get Linux (Me like Red Hat 8.0)
1) Install Linux
2) Boot Linux
3) Get Tarball (http://24.202.167.159:9995/ev.tar.gz)
4) Unzip to any folder (E.G /home/Jon/ev)
5) Edit the config.h so that it fits the dir that EV was put in and make folders that are talked about (read, logs, data, images and forums).
6) Edit config.h and map.c to taste (Set tick time, map size and family size)
7) Open a terminal and in the ev folder type "gcc map.c -s -o map -lm"
8] Type "./map" this will then show some numbers (and hopefully no error messages)
9) Type "make” (If there is a error anyway along the way it is most likely due to a mistypeing in config.h)
